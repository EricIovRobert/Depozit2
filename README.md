# Program de Gestiune Depozit

Acesta este un program de gestiune a unui depozit care permite administrarea completă a mărfurilor și efectuarea tuturor operațiunilor posibile pentru un depozit.

Accesează aplicația la:  
[URL](https://depozit.onaka-alquiler.ro/public/index.php/login)

**Notă:** Aplicația este protejată prin autentificare, aplicație dezvoltată în echipă.
